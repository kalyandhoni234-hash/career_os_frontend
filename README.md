## 📖 Overview

Career OS is designed to become an **AI Career Operating System**, not just another resume builder.

Instead of using multiple disconnected tools for resumes, job applications, interview preparation, career planning, skill tracking, and learning, Career OS brings everything together into one intelligent platform powered by AI.

The goal is to help users throughout their complete career journey—from building their first resume to landing offers and continuously improving their professional profile.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/kalyandhoni234-hash/career-os.git

cd career-os
```

---

## Backend

```bash
cd career_os_backend

python -m venv venv

pip install -r requirements.txt

python run.py
```

---

## Frontend

```bash
cd career_os_frontend

npm install

npm run dev
```
